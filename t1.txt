========================================
ANDROID PROGRAM FILE COLLECTION
========================================

Description:
This repository contains various Android Studio example programs.
Some files may appear repeated with an extra letter at the end
(example: image.txt and imagee.txt).

These repeated files are usually:
• Updated versions
• Modified examples
• Alternative question solutions
• Corrected versions of previous files

Use the latest or updated file wherever necessary.
